# Dividi Item

Mini app/PWA per telefono.

## Funzioni

- Aggiungi persone
- Aggiungi item
- Salvataggio locale con `localStorage`
- Selezione degli item da dividere
- Divisione casuale equa
- Condivisione del risultato su WhatsApp

## Come usarla in locale

Apri `index.html` nel browser.

Per testare meglio la PWA:

```bash
python -m http.server 8000
```

Poi apri:

```text
http://localhost:8000
```

## Pubblicazione consigliata

Carica la cartella su GitHub Pages.

Sul telefono potrai aprire il link, poi usare:

- Android/Chrome: menu → Aggiungi a schermata Home
- iPhone/Safari: condividi → Aggiungi alla schermata Home

## Nota WhatsApp

Il messaggio viene inviato dal tuo numero perché l'app apre WhatsApp sul tuo telefono.
Per motivi di sicurezza, l'invio finale richiede sempre una conferma manuale dentro WhatsApp.
